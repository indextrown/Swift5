## 📝 [Concurrency] GCD란?

### 1. 기존 동시성 프로그래밍(GCD)

- 대기열에 넣어서 2번 cpu에 일을 시키는 방식으로 비동기 처리(대기열/큐 사용)

```swift
// 기존 동시성 프로그래밍(GCD)
// 다른 쓰레드로 보내지 않는다면 1번 쓰레드에서 실행된다
print("메인 쓰레드에서 실행 - 1")
print("메인 쓰레드에서 실행 - 2")

// 무조건 2번(또는 특정의 백그라운드) 스레드로 보내서 실행하겠다는 의미 
// (클로저 내에서는 순차적 실행)
DispatchQueue.global().async {
    print("백그라운드 쓰레드에서 실행 - 1")
    print("백그라운드 쓰레드에서 실행 - 2")
}

print("메인 쓰레드에서 실행 - 3")
print("메인 쓰레드에서 실행 - 4")
```
